# Microsoft SC-200 Labs

Hands-on Microsoft Security Operations Analyst (SC-200) lab environment documenting my practical experience while preparing for the **Microsoft Certified: Security Operations Analyst Associate (SC-200)** certification.

This repository contains practical Microsoft Sentinel and Microsoft Defender XDR labs covering threat detection, incident investigation, advanced hunting, Kusto Query Language (KQL), analytics rules, watchlists, workbooks, and Security Operations Center (SOC) workflows.

---

## Project Objective

The primary goal of this repository is to strengthen my Security Operations and threat hunting skills through hands-on practice rather than theory alone.

Each lab focuses on a specific Microsoft Sentinel or Microsoft Defender XDR capability and includes:

- Practical Microsoft Sentinel implementations
- Step-by-step documentation
- Screenshots
- Personal notes and lessons learned
- Kusto Query Language (KQL) queries
- Detection rule creation
- Threat hunting scenarios
- Git version history

## Final Project

The Final Project combines multiple Microsoft Sentinel and Microsoft Defender XDR concepts into a complete Security Operations Center (SOC) demonstration.

Key components include:

- Microsoft Sentinel Workspace
- Advanced Hunting with KQL
- Custom Detection Rules
- Watchlists
- Workbooks
- Microsoft Defender XDR Integration
- Security Monitoring
- Threat Investigation Workflow

This project demonstrates how Microsoft Sentinel can be used as a cloud-native SIEM and SOAR platform to investigate security events, perform threat hunting, and visualize security data through workbooks and dashboards.
