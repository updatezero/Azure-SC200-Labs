# Lab 02 - Connect Microsoft Defender XDR to Microsoft Sentinel

## Overview

This lab demonstrates how to extend Microsoft Sentinel by installing the Microsoft Defender XDR solution from the Content Hub. The deployment adds built-in analytics rules, hunting queries, workbooks and data connector integration required for modern Security Operations.

---

## Objectives

- Access the Microsoft Sentinel Content Hub
- Install the Microsoft Defender XDR solution
- Review deployed solution components
- Validate available data connectors
- Verify successful deployment

---

## Deployment Steps

1. Open Microsoft Sentinel.
2. Navigate to Content Hub.
3. Search for Microsoft Defender XDR.
4. Install the solution.
5. Verify successful installation.
6. Review installed solution components.
7. Open Data Connectors.
8. Validate connector availability.
