# Lab 01 - Deploy Microsoft Sentinel

## Overview

This lab demonstrates the deployment of Microsoft Sentinel using an Azure Log Analytics Workspace. It establishes the foundation for future Security Operations (SOC) labs by preparing the SIEM environment.

---

## Objectives

- Create a Resource Group
- Deploy a Log Analytics Workspace
- Enable Microsoft Sentinel
- Verify successful deployment
- Prepare the environment for future SC-200 labs

---

## Azure Services Used

- Microsoft Sentinel
- Azure Resource Group
- Azure Log Analytics Workspace

---

## Architecture

Azure Subscription
