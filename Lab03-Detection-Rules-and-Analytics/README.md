# Lab 03 – Microsoft Sentinel Analytics Rules

## Objective

The goal of this lab was to explore Microsoft Sentinel Analytics Rules within the Microsoft Defender portal. I reviewed built-in rule templates, examined rule configuration settings, and walked through the complete analytics rule creation wizard without deploying a production rule.

---

## Technologies

- Microsoft Sentinel
- Microsoft Defender XDR
- Kusto Query Language (KQL)
- Analytics Rules
- MITRE ATT&CK
- Incident Management

---

## Tasks Completed

- Opened the Analytics workspace
- Reviewed built-in Analytics Rule Templates
- Inspected rule metadata
- Examined KQL detection logic
- Started the Analytics Rule Wizard
- Reviewed Rule Logic configuration
- Reviewed Incident Settings
- Reviewed Automated Response configuration
- Reviewed final rule summary before deployment
