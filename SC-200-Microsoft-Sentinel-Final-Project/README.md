# Final Project – Microsoft Sentinel Enterprise SOC

## Overview

This final project demonstrates the deployment and exploration of Microsoft Sentinel as a cloud-native SIEM and SOAR platform.

The objective was to build a small Security Operations Center (SOC) environment by configuring Microsoft Sentinel, connecting data sources, running Kusto Query Language (KQL) queries, creating custom detections, using Watchlists, and exploring built-in Workbooks.

---

## Technologies Used

- Microsoft Sentinel
- Microsoft Defender XDR
- Azure Monitor
- Log Analytics Workspace
- Kusto Query Language (KQL)
- Watchlists
- Workbooks
- Microsoft Defender for Endpoint

---

## Skills Demonstrated

- Deploying Microsoft Sentinel
- Exploring Security Events using KQL
- Advanced Hunting
- Creating Custom Detection Rules
- Using Watchlists
- Exploring Sentinel Workbooks
- Basic SOC investigation workflow
- Microsoft Defender XDR integration
