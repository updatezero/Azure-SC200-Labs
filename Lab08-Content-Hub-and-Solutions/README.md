# Lab 08 – Microsoft Sentinel Workbooks

## Objective

The objective of this lab was to explore Microsoft Sentinel Workbooks and understand how built-in workbook templates provide visual dashboards for monitoring security data.

## Tasks Completed

- Opened Microsoft Sentinel Workbooks.
- Explored available workbook templates.
- Reviewed the Microsoft Defender for Endpoint workbook template.
- Opened the workbook preview.
- Navigated through available workbook tabs (Overview, Device, Network, File).
- Verified workbook structure and visualization components.

## Observations

The workbook loaded successfully, but the environment contained no Microsoft Defender telemetry. As a result, the visualizations displayed "The query returned no results."

This is expected in a lab environment where no endpoint data has been ingested.
