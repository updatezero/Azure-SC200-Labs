# Notes – Lab 08

## What are Workbooks?

Microsoft Sentinel Workbooks are interactive dashboards that visualize security data using:

- KQL queries
- Charts
- Tables
- Metrics
- Maps
- Time-based visualizations

## Benefits

- SOC dashboards
- Incident monitoring
- Threat hunting
- Device monitoring
- Executive reporting

## Workbook Templates

Microsoft provides built-in templates for:

- Microsoft Defender for Endpoint
- Microsoft Defender for Identity
- Microsoft Defender for Office 365

The workbook structure was still fully accessible and demonstrated how dashboards are organized.
